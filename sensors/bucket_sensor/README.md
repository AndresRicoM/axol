<h2>Programming</h2>

We need a [USB-TTL adapter](https://duckduckgo.com/?q=USB-TTL+adapter&atb=v313-1&iax=images&ia=images) to program the sensor. 

Connect Cables as Follows:

```
Bucket Sensor - Adapter
3.3V --- 3.3V (or 5V)
GND  --- GND
TX   --- RX
RX   --- TX
```
